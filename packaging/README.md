# Packaging Files

This directory contains files needed for distribution packaging.

## Files

- `cachyos-service-manager-qt.desktop` - Desktop entry for Qt6 version
- `cachyos-service-manager-gtk.desktop` - Desktop entry for GTK4 version
- `cachyos-service-manager.png` - Application icon (256x256)

## Icon

The application icon should be placed here as `cachyos-service-manager.png`.

You can download the generated icon from:
https://user-gen-media-assets.s3.amazonaws.com/seedream_images/c0425dd5-e105-472e-807c-c00585c28b06.png

Or create your own 256x256 PNG icon with teal/blue CachyOS branding.
